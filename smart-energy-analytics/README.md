# Практична робота 13.1: Time-Series Analytics (Варіант 1)

## Опис архітектури проєкту

Проєкт демонструє збір та аналітику IoT даних за допомогою часових баз даних (Time-Series Databases) у середовищі OpenHAB.

## Реалізований функціонал:

1. **Збереження даних (Persistence)**: Налаштовані стратегії запису даних при зміні стану та періодично кожну хвилину (`persistence/rrd4j.persist`).
2. **Агрегація даних (Analytics Rules)**: Складено правила для підрахунку сумарного навантаження, середнього значення за годину та добових максимумів.
3. **Візуалізація (Dashboard)**: Побудовано графіки (Charts) для відображення трендів споживання у часі.

## Структура папок:

- `items/` — визначення джерел даних та метрик.
- `persistence/` — стратегії збереження часових рядів.
- `rules/` — аналітичні обчислення (агрегація).
- `sitemaps/` — графічний інтерфейс користувача з графіками.
